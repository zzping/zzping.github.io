# Mybatis 插件机制

# 插件简介

MyBatis 允许用户在已映射语句执行过程中的某一点进行拦截调用。MyBatis 使用插件来拦截的方法调用，故此MyBatis 插件通常称为：Mybatis 拦截器。默认情况下，MyBatis 允许使用插件来拦截 Mybatis 中的四大对象

  • Executor
  • ParameterHandler
  • ResultSetHandler
  • StatementHandler

说明：Mybatis 可以对这四个接口中所有的方法进行拦截。

# 自定义插件

（1）编写自定义类，实现 Interceptor 接口， 并通过 @Interceptor 注解完成插件签名

// 插件签名: 告诉 Mybatis 当前插件用来拦截哪个对象的哪个方法
@Intercepts({
    @Signature(type = StatementHandler.class,method = "parameterize", args = java.sql.Statement.class)
})
public class MyFirstPlugin implements Interceptor {

    // 拦截目标对象的目标方法的执行
    @Override
    public Object intercept(Invocation invocation) throws Throwable {
        System.out.println("【MyFirstPlugin】...intercept()..., 实际拦截的目标对象: " + invocation.getTarget() + ", 目标方法: " + invocation.getMethod());

        // 在执行 StatementHandler 的 parameterize 方时修改 sql 的参数
        MetaObject metaObject = SystemMetaObject.forObject(invocation.getTarget());
        Object value = metaObject.getValue("parameterHandler.parameterObject");
        System.out.println("sql语句使用的参数是: " + value);
        metaObject.setValue("parameterHandler.parameterObject", 2);

        // 执行目标方法并返回
        return invocation.proceed();
    }

    // 包装目标对象: 为目标对象创建代理对象
    @Override
    public Object plugin(Object target) {
        System.out.println("【MyFirstPlugin】...plugin()..., mybatis 将要包装的对象: " + target);
        // 返回包装后的对象(底层是JDK的动态代理)
        return Plugin.wrap(target,this);
    }

    // 将注册插件时的 properties 属性设置进来
    @Override
    public void setProperties(Properties properties) {
        System.out.println("【MyFirstPlugin】...setProperties()...");
        System.out.println("插件配置的属性: " + properties);
    }
}

# 分页插件 PageHelper

# （1）分页知识回顾

在实际的项目开发中，基本都会用到分页，分页可以分为： 物理分页、逻辑分页

  • 物理分页： 直接查询对应分页的数据，依赖数据库的支持，MySQL 通过 limit 关键字支持来物理分页。

  • 逻辑分页： 一次性查询所有数据，通过程序代码中获取分页数据，依赖程序。

Mybatis 自带的 RowBounds 支持逻辑分页，使用 SqlSession 的 API 传入 RowBounds 分页对象即可实现分页。

// Mybatis 自带的 RowBounds 逻辑分页测试
@Test
public void test01() {
   SqlSession sqlSession = sqlSessionFactory.openSession();
   RowBounds rowBounds = new RowBounds(1, 5);
   String statementId = "com.zzp.mapper.UserMapper.getUserList";
   List<Object> list = sqlSession.selectList(statementId, null, rowBounds);
   list.forEach(System.out::println);
}

# （2）PageHelper 简介

PageHelper 是一个开源的 Mybatis 分页插件。支持 12 种常见数据库的物理分页，支持多种分页方式。

官网地址： https://pagehelper.github.io/

Github 地址： https://github.com/pagehelper/Mybatis-PageHelper

Github 文档地址： https://github.com/pagehelper/Mybatis-PageHelper/blob/master/README_zh.md

# （3）PageHelper 使用

1、引入 maven 依赖

    <version>5.2.0</version>

3、使用分页插件 PageHelper

常用方式一：PageHelper.startPage 静态方法调用

@Test
public void test02() {
   SqlSession sqlSession = sqlSessionFactory.openSession();
   UserMapper mapper = sqlSession.getMapper(UserMapper.class);

   // PageHelper 静态方法设置 pageNum、pageSize
   Page<Object> page = PageHelper.startPage(2, 5);
   List<User> userList = mapper.getUserList();
   userList.forEach(System.out::println);

   System.out.println("------------------------------------------");
   System.out.println("总记录数: " + page.getTotal());
   System.out.println("总页数: " + page.getPages());
   System.out.println("当前页码: " + page.getPageNum());
   System.out.println("每页记录数: " + page.getPageSize());
}

常用方式二：使用 PageInfo 的方式

@Test
public void test03() {
   SqlSession sqlSession = sqlSessionFactory.openSession();
   UserMapper mapper = sqlSession.getMapper(UserMapper.class);
   // PageHelper 静态方法设置 pageNum、pageSize
   PageHelper.startPage(1,5);
   List<User> userList = mapper.getUserList();
   // 使用 PageInfo 包装查询返回的结果
   PageInfo<User> pageInfo = new PageInfo<>(userList);

   pageInfo.getList().forEach(System.out::println); // 分页列表数据

   System.out.println("------------------------------------------");
   System.out.println("总记录数: " + pageInfo.getTotal());
   System.out.println("总页数: " + pageInfo.getPages());
   System.out.println("当前页码: " + pageInfo.getPageNum());
   System.out.println("每页记录数: " + pageInfo.getPageSize());
   System.out.println("是否第一页: " + pageInfo.isIsFirstPage());
   System.out.println("是否最后一页: " + pageInfo.isIsLastPage());
}

# 插件总结

Mybatis 的插件实际上是通过拦截指定对象的指定方法，然后通过动态代理包装对应对象，完成插件的功能。
