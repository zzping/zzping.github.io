# Mybatis 使用的整体步骤

(1) 根据 xml 配置文件(全局配置文件) 创建一个 SqlSessionFactory 对象
(2) 从 SqlSessionFactory 中获取一个 SqlSession 实例
(3) 使用 SqlSession 对象来执行对数据库的增删改查
(4) 使用完成后关闭资源(SqlSession), 一个 SqlSession 就代表和数据库的一次会话, 用完要关闭

  `id` int(11) not null auto_increment comment '主键id',

（3）编写 UserMapper 接口对应的 SQL 映射文件，位于 resources/mapper/UserMapper.xml

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<!-- namespace 对应 Mapper 接口的全限定类名 -->
<mapper namespace="com.zzp.mapper.UserMapper">

    <!-- 根据 id 查询 User -->
    <select id="getUserById" resultType="com.zzp.entity.User">
        select id, last_name as lastName, gender, email
        from t_user
        where id = #{id}
    </select>
</mapper>

（4）编写 Mybatis 全局配置文件，位于 resources/mybatis-config.xml

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE configuration
        PUBLIC "-//mybatis.org//DTD Config 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-config.dtd">
<configuration>
    <environments default="development">
        <environment id="development">
            <transactionManager type="JDBC"/>
            <dataSource type="POOLED">
                <property name="driver" value="com.mysql.jdbc.Driver"/>
                <property name="url" value="jdbc:mysql://localhost:3306/mybatis01"/>
                <property name="username" value="root"/>
                <property name="password" value="root"/>
            </dataSource>
        </environment>
    </environments>

    <!-- 将 SQL 映射文件(UserMapper.xml) 注册到全局配置文件(mybatis-config.xml)中 -->
    <mappers>
        <mapper resource="mapper/UserMapper.xml"/>
    </mappers>
</configuration>

（6）编写测试程序

public class MybatisTest {

    @Test
    public void test01() throws IOException {
        // 1、使用 SqlSessionFactoryBuilder 根据全局配置文件创建 SqlSessionFactory 对象
        InputStream inputStream = Resources.getResourceAsStream("mybatis-config.xml");
        SqlSessionFactory sqlSessionFactory = new SqlSessionFactoryBuilder().build(inputStream);

        // 2、使用 SqlSessionFactory 获取 SqlSession, SqlSession 代表和数据库的一次会话
        SqlSession sqlSession = sqlSessionFactory.openSession();

        try {
            // 3、使用 SqlSession 获取 Mapper 接口(SQL映射器), Mybatis 会自动为接口创建一个代理对象
            UserMapper userMapper = sqlSession.getMapper(UserMapper.class);
            System.out.println(userMapper); // org.apache.ibatis.binding.MapperProxy@43738a82
            System.out.println(userMapper.getClass()); // class com.sun.proxy.$Proxy6
            User user = userMapper.getUserById(1);
            System.out.println(user);
        } finally {
            // 4、关闭 sqlSession
            sqlSession.close();
        }
    }
}

（6）小结

 (1) 两种重要的配置文件： 全局配置文件、SQL 映射文件
 (2) SqlSession 代表和数据库的一次会话, 用完需要关闭。
 (3) SqlSession 和 Connection 一样, 都是非线程安全的。每次使用都应该去获取新的实例对象。
 (4) Mapper 接口没有实现类, Mybatis 会为 Mapper 接口生成一个代理对象
