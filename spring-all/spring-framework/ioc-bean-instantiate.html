# 静态工厂方法实例化

public class UserStaticFactory {

    public static User getUser(String lastName) {
        System.out.println("UserStaticFactory.getUser()...");
        User user = new User();
        user.setLastName(lastName);
        user.setAge(20);
        user.setEmail("123@demo.com");
        user.setGender("男");
        return user;
    }
}

# 实现 FactoryBean 接口实例化

实现了 FactoryBean 接口的类是 Spring 可以认识的工厂类、Spring 会自动调用 getObject（）方法创建对象。

这种方式不管 Bean 是否是单例，在容器创建时都不会实例化这个 Bean，都是在获取的时候才实例化对应的 Bean。

public class UserFactoryBean implements FactoryBean<User> {

    @Override
    public User getObject() throws Exception {
        System.out.println("UserFactoryBean.getObject()...");
        User user = new User();
        user.setLastName("小李");
        user.setAge(20);
        user.setEmail("123@demo.com");
        user.setGender("男");
        return user;
    }

    @Override
    public Class<?> getObjectType() {
        return User.class;
    }

    @Override
    public boolean isSingleton() {
        return true;
    }
}
