# Stream 流

# Stream 流简介

Java8 中有两大最为重要的改变，第一个是 Lambda表达式；另外一个则是 Stream API（java.util.stream.*）；Stream是 Java8 中提供的处理集合的关键抽象概念，它可以指定你希望对集合进行的操作，可以执行非常复杂的查找、过滤、映射数据等操作。使用 Stream API 对集合数据进行操作，就类似使用 SQL 对数据库进行操作一样简单；也可以使用 Stream API 来并行执行操作。简而言之，Stream API 提供了一种高效且易于使用的数据处理的方式。

# Stream 操作的三个步骤

（1）创建 Stream：一个数据源（如：集合、数组），获取一个流

（2）中间操作：一个中间操作链，对数据源的数据进行处理

（3）终止操作（终端操作）：一个终止操作，执行中间操作链，并产生结果

# Stream 流的创建

（1）Collection 提供了两个创建集合的方法：1、stream()和 2、parallelStream()

（2）Arrays的静态方法：Arrays.stream(array)

（3）Stream类的静态方法：Stream.of()（eg：Stream stream = Stream.of("aa", "bb", "cc")）

（4）Stream类的静态方法创建无限流：Stream.iterate()、Stream.generate（eg：Stream.iterate(0, (x) -> x+2)）

@Test
public void test01() {
    // 1. Collection 接口提供了两个方法 stream()、parallelStream()
    List<String> list = new ArrayList<>();
    Stream<String> stream1 = list.stream(); // 获取一个顺序流
    Stream<String> stream2 = list.parallelStream(); // 获取一个并行流

    // 2. Arrays 的 stream() 获取一个数组流
    Integer[] nums = new Integer[10];
    Stream<Integer> stream = Arrays.stream(nums);

    // 3. 通过 Stream 的静态方法 of()
    Stream<Integer> integerStream = Stream.of(1, 2, 3, 4, 5);

    // 4. 创建无限流
    // 迭代
    Stream<Integer> stream3 = Stream.iterate(0, (x) -> x + 2).limit(100);
    stream3.forEach(System.out::println);

    // 生成
    Stream<Double> stream4 = Stream.generate(Math::random).limit(5);
    stream4.forEach(System.out::println);
}

# Stream 流的中间操作

（1）筛选与切片

filter： 接收Lambda，从流中排出某些元素
limit： 截断流，使其元素不超过给定数量
skip： 跳过元素，返回一个扔掉了前n个元素的流，若流中元素不足n个，则返回一个空流。与limit(n)互补
distinct： 去重筛选，通过流所生成的元素的 hashCode（）和equals（）方法去重元素

（2）映射

map：接收Lambda，将元素转换成其他形式或提取信息，接收一个函数作为参数，该函数会被应用到每一个元素上，并将其映射成一个新元素
flatMap：接收一个函数作为参数，将流中的每个值都换成另一个流，然后把所有流连接成一个流

// map 和 flatMap 的区别类似于集合的 add(List) 和 addAll(List) 的区别
@Test
public void test02() {
    // map
    employees.stream()
            .map(Employee::getName)
            .forEach(System.out::println);
    System.out.println("==============================================");

    // flatMap
    employees.stream()
            .flatMap(employee -> Stream.of(employee.getName(), employee.getAge()))
            .forEach(System.out::println);
    System.out.println("==============================================");

    // floatMap
    List<String> list = Arrays.asList("abc", "de", "fg");
    Stream<Character> characterStream = list.stream()
            .flatMap(TestStreamAPI03::filterCharacter);

    characterStream.forEach(System.out::println);
}

private static Stream<Character> filterCharacter(String str) {
    List<Character> list = new ArrayList<>();

    for (Character ch : str.toCharArray()) {
        list.add(ch);
    }

    return list.stream();
}

noneMatch()： 检查是否没有匹配的元素
findFirst()： 返回第一个元素
findAny()： 返回当前流中的任意元素
count()： 返回流中元素的总个数
max()： 返回流中最大值
min()： 返回流中最小值

collection(Collector collector)


    // collect()： 收集
    @Test
    public void test02() {
    // 收集为 List 集合

    // 收集为指定集合
    LinkedHashSet<String> collect = employees.stream().map(Employee::getName).collect(Collectors.toCollection(LinkedHashSet::new));

    employees.stream().map(Employee::getSalary).collect(Collectors.maxBy(Double::compare));

    // 收集最低的薪资
    employees.stream().map(Employee::getSalary).collect(Collectors.minBy((e1, e2) -> e1.compareTo(e2)));

    // 收集薪资总和
    Double salarySum = employees.stream().collect(Collectors.summingDouble(Employee::getSalary));

    // 收集数据的个数
    Long amount = employees.stream().collect(Collectors.counting());

    DoubleSummaryStatistics summaryStatistics = employees.stream().collect(Collectors.summarizingDouble(Employee::getSalary));

    double max = summaryStatistics.getMax();
    double min = summaryStatistics.getMin();
    double average = summaryStatistics.getAverage();

    // 分区: 分为薪资大于 5000 和不大于 5000 的
    Map<Boolean, List<Employee>> collect1 = employees.stream().collect(Collectors.partitioningBy(e -> e.getSalary() > 5000));

# Stream API 的练习

/**
 * Stream API 练习
 */
public class TestStreamAPI05 {

    private List<Employee> employees = Arrays.asList(
        new Employee(199, "张三", 32, 7777.99),
        new Employee(3, "李四", 32, 4567.99),
        new Employee(25, "王五", 42, 8888.99),
        new Employee(59, "赵六", 38, 9999.99),
        new Employee(120, "郑七", 19, 4096.99)
    );

    /*
      1. 给定一个数字列表，如何返回一个由每个数的平方构成的列表呢？
          给定【1，2，3，4，5】， 应该返回【1，4，9，16，25】。
    */
    @Test
    public void test01() {
        Integer[] nums = new Integer[]{1,2,3,4,5};
        Arrays.stream(nums)
            .map(e -> e * e)
            .forEach(System.out::println);
    }

    /*
    2.    怎样用 map 和 reduce 方法数一数流中有多少个 Employee 呢？
    */
    @Test
    public void test02() {
        Integer count = employees.stream()
            .map(e -> 1)
            .reduce(0, Integer::sum);

        System.out.println(count);
    }


    //=================================交易相关的集合使用 Stream 操作【Transaction、Trader】============================
    List<Transaction> transactions = null;

    @Before
    public void before(){
        Trader raoul = new Trader("Raoul", "Cambridge");
        Trader mario = new Trader("Mario", "Milan");
        Trader alan = new Trader("Alan", "Cambridge");
        Trader brian = new Trader("Brian", "Cambridge");

        transactions = Arrays.asList(
            new Transaction(brian, 2011, 300),
            new Transaction(raoul, 2012, 1000),
            new Transaction(raoul, 2011, 400),
            new Transaction(mario, 2012, 710),
            new Transaction(mario, 2012, 700),
            new Transaction(alan, 2012, 950)
        );
    }

    // 1. 找出2011年发生的所有交易， 并按交易额排序（从低到高）
    @Test
    public void test03() {
        transactions.stream()
            .filter(e -> e.getYear() == 2011)
            .sorted(Comparator.comparing(Transaction::getValue))
            .forEach(System.out::println);
    }

    // 2. 交易员都在哪些不同的城市工作过？
    @Test
    public void test04() {
        transactions.stream()
            .map(e -> e.getTrader().getCity())
            .distinct()
            .forEach(System.out::println);
    }

    // 3. 查找所有来自剑桥的交易员，并按姓名排序
    @Test
    public void test05() {
        transactions.stream()
            .filter(e -> e.getTrader().getCity().equals("Cambridge"))
            .map(Transaction::getTrader)
            .sorted(Comparator.comparing(Trader::getName))
            .distinct()
            .forEach(System.out::println);
    }

    // 4. 有没有交易员是在米兰工作的？
    @Test
    public void test06() {
        boolean flag = transactions.stream()
            .anyMatch(e -> e.getTrader().getCity().equals("Milan"));
        System.out.println(flag);
    }

    // 5. 打印生活在剑桥的交易员的所有交易额
    @Test
    public void test07() {
        Integer sum = transactions.stream()
            .filter(e -> e.getTrader().getCity().equals("Cambridge"))
            .map(Transaction::getValue)
            .reduce(0, Integer::sum);
        System.out.println(sum);
    }

    // 6. 所有交易中，最高的交易额是多少
    @Test
    public void test08() {
        Optional<Integer> max = transactions.stream()
            .map(Transaction::getValue)
            .max(Integer::compareTo);
        System.out.println(max.get());
    }

    // 7. 找到交易额最小的交易
    @Test
    public void test09() {
        Optional<Transaction> min = transactions.stream()
            .min(Comparator.comparing(Transaction::getValue));
        System.out.println(min.get());
    }

    // 8. 返回所有交易员的姓名字符串，按字母顺序排序
    @Test
    public void test10() {
        transactions.stream()
            .map(e -> e.getTrader().getName())
            .distinct()
            .sorted()
            .forEach(System.out::println);

        System.out.println("============================================");

        String str = transactions.stream()
            .map(e -> e.getTrader().getName())
            .reduce("", (e1, e2) -> e1.concat(e2));// .reduce("", String::concat);
        System.out.println(str);

        System.out.println("============================================");

        transactions.stream()
            .map(e -> e.getTrader().getName())
            .flatMap(TestStreamAPI05::filterCharater)
            .sorted(String::compareToIgnoreCase)
            .forEach(System.out::print);

    }

    private static Stream<String> filterCharater(String string){
        List<String> list = new ArrayList<>();

        for (Character c : string.toCharArray()) {
            list.add(c.toString());
        }

        return list.stream();
    }
}
