# 函数式接口

# Lambda 表达式需要函数式接口的支持

Lambda 表达式实际上是对接口的实现【确切地说是函数式接口】。所以要使用 Lambda 表达式首先要有一个函数式接口。因此 Java8 中提供了许多的函数式接口供使用。

# Java8 中提供内置的函数式接口

Java8 中内置的四大核心的函数式接口，都位于 java.util.function 包下

# （1）四大内置的函数式接口

// 消费型接口
@FunctionalInterface
public interface Consumer<T> {

    void accept(T t);
} 

// 供给型接口
@FunctionalInterface
public interface Supplier<T> {

    T get();
}

// 函数型接口
@FunctionalInterface
public interface Function<T, R> {

    R apply(T t);
}

// 断言型接口
@FunctionalInterface
public interface Predicate<T> {

    boolean test(T t);
}

下面通过一些代码来测试这些函数式接口的使用：

/**
 * 测试 Java8 中内置的四大核心的函数式接口, 位于 java.util.function 包下
*/
public class TestLambda02 {

    // 测试消费型接口 Supplier<T>
    @Test
    public void test01() {
        happy(3000, (e) -> System.out.println("今天消费了 " + e + " 元"));
    }

    private void happy(double money, Consumer<Double> con) {
        con.accept(money);
    }

    // 测试供给型接口 Supplier<T>
    @Test
    public void test02() {
        List<Integer> numList = getNumList(5, () -> (int) (Math.random() * 100));
        numList.forEach(System.out::println);
    }

    // 产生指定个数的整数，并放入集合中
    private List<Integer> getNumList(int num, Supplier<Integer> supplier) {
        List<Integer> list = new ArrayList<>();
        for (int i = 0; i < num; i++) {
            Integer number = supplier.get();
            list.add(number);
        }
        return list;
    }

    // 测试函数型接口 Function<T, R>
    @Test
    public void test03() {
        String str1 = strHandler("\t\t\t 新年快乐！", (e) -> e.trim());
        System.out.println(str1);

        String str2 = strHandler("i love you!", e -> e.toUpperCase());
        System.out.println(str2);
    }

    // 用于处理字符串
    private String strHandler(String str, Function<String, String> fun) {
        return fun.apply(str);
    }

    // 测试断言型接口  Predicate<T>
    @Test
    public void test04() {
        List<String> list = Arrays.asList("Html", "css", "js", "Tomcat", "SpringBoot");
        List<String> stringList = filterStr(list, e -> e.length() > 3);
        stringList.forEach(System.out::println);
    }

    // 将满足条件的字符串放入集合中
    private List<String> filterStr(List<String> list, Predicate<String> pre) {
        List<String> stringList = new ArrayList<>();
        for (String s : list) {
            if (pre.test(s)) {
                stringList.add(s);
            }
        }
        return stringList;
    }
}
