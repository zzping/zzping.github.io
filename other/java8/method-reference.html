# 方法引用的使用

（1）什么时候使用方法引用？当要传递给Lambda体的操作，已经有实现的方法了，就可以使用方法引用

（2）使用方法引用的前提：Lambda 体中的方法的参数列表和返回值必须要与函数式接口中的抽象方法的参数列表和返回值保持一致

（3）方法引用可以理解为是 Lambda 表达式的另外一种表达形式，它让 Lambda 表达式更加简单

（4）方法引用使用操作符::将方法名和对象或类名分割开来，主要有以下三种语法格式：

  • 对象::实例方法
  • 类::静态方法
  • 类::实例方法
@Test
public void test01() {
    PrintStream out = System.out;
    Consumer<String> con = (e) -> out.println(e);
    con.accept("hello baby");

    // 使用方法引用后
    Consumer<String> con1 = out::println;
    con1.accept("hello baby");
}

// 对象的引用::实例方法名
@Test
public void test02() {
    Employee employee = new Employee(101, "ldd", 23, 9000.0);
    Supplier<String> sup = () -> employee.getName();
    System.out.println(sup.get());

    // 使用方法引用后
    Supplier<String> sup1 = employee::getName;
    System.out.println(sup1.get());
}

// 类名::静态方法名
@Test
public void test03() {
    BiFunction<Double, Double, Double> bif = (x, y) -> Math.max(x, y);
    System.out.println(bif.apply(100.01, 99.25));

    // 使用方法引用后
    BiFunction<Double, Double, Double> bif2 = Math::max;
    System.out.println(bif2.apply(2.3, 4.5));

    System.out.println("=====================================");
    Comparator<Integer> comparator = (e1, e2) -> e1.compareTo(e2);
    Comparator<Integer> com = (e1, e2) -> Integer.compare(e1, e2);
    // 静态方法 compare
    Comparator<Integer> com1 = Integer::compare;
    // 实例方法 compareTo【compareTo 实际上也还是调用的 compare 方法】
    Comparator<Integer> com2 = Integer::compareTo;
}

// 类名::实例方法名
@Test
public void test04() {
    BiPredicate<String, String> bp = (x, y) -> x.equals(y);
    System.out.println(bp.test("abcde", "abcde"));

    System.out.println("-----------------------------------------");

    BiPredicate<String, String> bp2 = String::equals;
    System.out.println(bp2.test("abc", "abc"));

    System.out.println("-----------------------------------------");


    Function<Employee, String> fun = (e) -> e.show();
    System.out.println(fun.apply(new Employee()));

    System.out.println("-----------------------------------------");

    Function<Employee, String> fun2 = Employee::show;
    System.out.println(fun2.apply(new Employee()));
}

# 构造器引用的使用

（1）语法格式：类名 :: new

// 构造器引用
@Test
public void test05() {
    Supplier<Employee> sup = () -> new Employee();
    System.out.println(sup.get());

    // 使用构造器引用之后
    Supplier<Employee> sup2 = Employee::new;
    System.out.println(sup2.get());

}
